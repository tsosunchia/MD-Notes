### •认识异或运算

异或运算：相同为0，不同为1

同或运算：相同以1，不同为0

能长时间记住的概率接近0%

所以，异或运算就记成无进位相加！

### 异或运算的性质

1)0^N == N   N^N == 0

2)异或运算满足交换律和结合率

上面的两个性质，用无进位相加来理解就非常的容易
